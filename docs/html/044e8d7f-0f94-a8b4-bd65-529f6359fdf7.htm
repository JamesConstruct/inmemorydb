      InMemoryDB Namespace

      Jednoduchá **in-memory** databáze napsaná v C# jako zápočtový projekt do Programování v C#. Databáze má polymorfní strukturu a může obsahovat libovolný počet sloupců (omezení pamětí) různého druhu. Databáze nativně podporuje základní datové typy C#, avšak je možné ji snadno rozšířit tak, aby pracovala s jakýmkoli typem implementujícím IComparable a IEquitable interface. Databáze podporuje indexování a binární vyhledávání v logaritmickém čase, stejně jako pokročilé možnosti filtrování záznamů.

      Classes

      BooleanColumn Class representing a column of only boolean values. Supports logic operators like AND, OR (|) or NOT (!).
      ColumnT Generic class for column (internally a list of T values with added functionality for transformations and comparisons).
      Db The main in-memory database object.
      DbRecordWrapper Class that wraps around the Record and allows for dynamic access of values by column names (based on the columns of the underlying database).
      FieldT Field with value of the given type T.
      ParentField Abstract class for polymorphism.
      Program Program for demonstration of the database class.

      Interfaces

      IColumnT Column represents an entire column from table - list of values of the same type accross all rows in the database.
      ITreeT Interface for general search tree.
